PWA Icons Information

Required Icons

For this PWA to work properly, the following icon files need to be created:

  • favicon.ico - Site favicon (16x16, 32x32, 48x48)
  • apple-touch-icon.png - Apple touch icon (180x180)
  • pwa-192x192.png - PWA icon (192x192)
  • pwa-512x512.png - PWA icon (512x512)
  • masked-icon.svg - Safari pinned tab icon

Icon Creation

You can create these icons using:

  • Graphic design software like Adobe Illustrator, Photoshop, or GIMP
  • Online icon generators like RealFaviconGenerator
  • PWA asset generators like vite-pwa/assets-generator

File Locations

All icon files should be placed in the public directory of your project, so they can be properly referenced by the PWA manifest.
